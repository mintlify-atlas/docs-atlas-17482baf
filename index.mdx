<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-bold dark:text-white text-gray-900 mb-4">Quick start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get your development environment ready in minutes</p>

  <Steps>
    <Step title="Install Nix">
      Install Nix package manager with flakes support:
      
      ```bash
      curl -L https://nixos.org/nix/install | sh
      mkdir -p ~/.config/nix
      echo 'experimental-features = nix-command flakes' >> ~/.config/nix/nix.conf
      ```
      
      <Note>Windows users should install WSL first and run all commands inside WSL.</Note>
    </Step>

    <Step title="Clone and enter devshell">
      Clone the repository and enter the development environment:
      
      ```bash
      git clone https://github.com/Oyasaiserver/platform.git
      cd platform
      nix develop
      ```
      
      The devshell includes Java 25, Gradle 9, and all required build tools.
    </Step>

    <Step title="Build all plugins">
      Build all Minecraft plugins in parallel:
      
      ```bash
      gradle build --parallel
      ```
      
      Built plugins will be available in `plugins/*/build/libs/`.
    </Step>

    <Step title="Run a test server">
      Launch a minimal test server with your plugins:
      
      ```bash
      nix run .#oyasai-minecraft-minimal
      ```
      
      The server will start on `localhost:25565` with all custom plugins loaded.
    </Step>
  </Steps>
</div>
